## What Each File Does

- `index.html` — main public page
- `more_cases_under_development.html` — secondary placeholder page
- `README.md` — human overview of the project
- `.gitignore` — local repo hygiene
- `docs/project-context.md` — short memory file for future sessions
- `docs/run-commands.md` — practical usage notes
- `docs/site-structure.md` — quick orientation for the project layout
