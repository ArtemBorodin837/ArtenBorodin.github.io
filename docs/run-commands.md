# Run Commands

## Open Locally

Simplest option:

1. open `index.html` directly in a browser.

## Live Preview In VS Code

If Live Server is installed:

1. open `index.html`;
2. choose `Open with Live Server`.

## Git Basics For This Project

If this folder is later turned into a separate Git repository, the basic flow is:

```powershell
git status
git add .
git commit -m "Describe the change"
git push origin main
```

## Publishing

If GitHub Pages is connected to the repository:

1. edit files;
2. commit changes;
3. push to GitHub;
4. wait for GitHub Pages to refresh.
